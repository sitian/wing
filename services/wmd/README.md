**client1@192.168.1.101:**

**client2@192.168.1.102:**

path/to/wmd/gen.py 192.168.1.103

path/to/wmd/mds.py 192.168.1.1

**server2@192.168.1.2:**
path/to/wmd/mds.py 192.168.1.2

**server3@192.168.1.3:**
